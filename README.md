1. Install Microsoft Visual Studio Community 2022 (64-bit) edition and  PostgreSQL 18.3 version.
2. Create database with the name of WorkTrackLiteDB.
3. Restore the  Database file(WorkTrackLiteDB.backup) from the path https://github.com/pammibalajikumar/WorkTrackLite/tree/master/Database.
4. Update connection string in https://github.com/pammibalajikumar/WorkTrackLite/blob/master/Services/DatabaseService.cs in
the lineno-10  file with your postgresql username and password.
5. Run the application.
